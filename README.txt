
<h2>Musischool</h2>
Este es el código HTML de la página web de Musischool, una escuela de música contemporánea. El código proporciona la estructura y los elementos necesarios para visualizar el contenido de la página, así como interactuar con ella.
Estructura del proyecto
El proyecto se compone de los siguientes archivos y directorios:
index.html: Este archivo contiene la estructura principal del documento HTML.
assets/css/style.css: Este archivo contiene los estilos CSS que se aplican a la página.
assets/js/script.js: Este archivo contiene el código JavaScript utilizado en la página.
assets/video/pexels-thirdman-6194007-1280x720-25fps.mp4: Este archivo de video se reproduce en la sección "Nosotros" de la página.
assets/fonts/: Este directorio contiene las fuentes utilizadas en la página.
assets/images/: Este directorio contiene las imágenes utilizadas en la página.
Dependencias externas
El proyecto utiliza las siguientes dependencias externas:
Font Awesome: Se utiliza para los iconos en la página. La biblioteca se carga desde https://kit.fontawesome.com/49f8c9111c.js.
Mapbox: Se utiliza para mostrar un mapa interactivo en la página. La biblioteca se carga desde https://api.mapbox.com/mapbox-gl-js/v2.4.1/mapbox-gl.js. También se carga una hoja de estilos CSS relacionada desde https://api.mapbox.com/mapbox-gl-js/v2.4.1/mapbox-gl.css.
